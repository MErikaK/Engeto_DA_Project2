A Journey into the Booklover's World

OVERVIEW

This project explores the world of books through data analysis and visualization. The goal is to uncover patterns in book publishing, popularity, and reading preferences across time and regions, with a special focus on Czech books and global trends.
The interactive Power BI dashboard "A Journey into the Booklover's World" presents insights across several thematic sections, helping users understand how books are published, bought, and loved worldwide.

DATA SOURCES
All data were collected between 1st and 8th February 2025 from the following sources:
- Czechoslovak bibliographic database
- Kaggle datasets:
  Most Sold Books in the World
  Historical Book Publishing Trends (1500–2010)
- Book Recommendation API (GoodBooks API)
  
TOOLS  & TECHNOLOGIES
Power BI – data visualization and dashboard creation
Data preprocessing – (Power Query / Excel / DBeaver)
Kaggle datasets – external structured data sources
API integration – book recommendation data

DASHBOARD STRUCTURE 
The Power BI report is organized into four main sections:

1. Books We Love
Insights into popular and highly rated books
Most frequently recommended titles
Patterns in reader preferences

3. Books We Buy
Analysis of the most sold books globally
Commercial success vs. popularity
Trends in book purchases

5. 🇨🇿 Czechs in the Bookshelf
Focus on Czech literature and authors
Representation of Czech books in global context
Bibliographic insights from Czech sources

7. How WE Publish
Historical publishing trends (1500–2010)
Evolution of book production over time
Long-term industry dynamics

KEY INSIGHTS
Evolution of publishing shows significant growth after the industrial revolution and digital era
A small number of titles dominate global sales
There are clear differences between what people read, buy, and recommend
Czech literature has a distinctive presence when analyzed in bibliographic data
